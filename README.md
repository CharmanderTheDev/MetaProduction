MetaProduction is a new kind of automation game focused around near-infinite scalability. The core idea is that many large, computationally intensive builds in other factory-building games can be reduced to a functionally equivalent, much simpler input-output formula. For example, an enormous circuit manufacturing block in the game factorio could be reduced to a single, large building with a rate of input consumption and output production, enormously saving on computation and allowing for much larger bases. The goal of this repository is to implement MetaProduction using accelerated simulation and curve fitting. Essentially, players will be able to simulate a large build at high speeds with idealized inputs, and using the data from that simulation, the build will be transformed into a single entity with extremely simple rules. Of course, this prohibits the inclusion of certain complexities present in many other automation games, but for now those complexities will be simply avoided, until the potential introduction of more complex reduction systems beyond mere simulation.
